## Open AI TTS implementation for Microsoft Graph API 

Using Open AI model gpt-4o-mini-tts to convert pages from oneNote using GraphAPI


### Dependencies
```
msgraph-core
python-dotenv
openai==2.7.1
pydub==0.25.1
transformers==4.57.1
vllm
snac
```


### Start Up
```
export VLLM_CPU_KVCACHE_SPACE=8
```
